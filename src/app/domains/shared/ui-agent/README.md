# AG-UI for Angular

This folder contains a simple AG-UI integration for Angular.

It is intentionally limited to the concepts shown in this training and is not meant as a complete or universal AG-UI framework. At the same time, it can serve as a starting point for your own implementation and further development.

In this training, this implementation is mostly used as a black box. For a better understanding of selected concepts, parts of it are also discussed as a gray box.
